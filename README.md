### Telegram-bot
#### 0. Описание задания
```text
Меню
У бота три основные кнопки меню:

     1. Оставить заявку. 
        При нажатии на эту кнопку бот должен задать пользователям 3 вопроса. 
        1.1. Какое направление вашего бизнеса?
            • Продажа
            • Производство
            • Оказание услуг
        1.2. На какой платформе вы хотите разработать чат-бот?
            - Телеграмм 
            - Ватсап
            - Вайбер
        1.3. Какой у вас бюджет? От … До …
        После ответа на все вопросы бот должен запрашивать номер телефона. 
        И данная заявка со всеми ответами и номером должна отправляться администратору.
    
     2. Купить товар.
        Для реализации данного пункта вам нужно будет подключить тестовую платежку Юkassa. 
        После нажатия на данную кнопку бот должен предлагать выбор.
        2.1. Купить 1 раз
        2.2. Купить 2 раза
        При нажатии на любую из этих кнопок бот должен отправлять на оплату в Юкассу. 
        Тестовые реквизиты отправить мне в Телеграмм. 
        После оплаты пользователю должны начисляться условные единицы в пункте меню три. 
        
     3. Мой баланс. 
        При нажатии на данную кнопку бот должен отправлять пользователю сообщение:  “У вас … условных единиц”. 
        Количество зависит от того, сколько он купил.
       
Функции админа:
     У админа помимо основного функционала должна быть кнопка “ Отправить сообщение пользователям”. 
     При нажатии на нее бот должен запрашивать сообщение и отправлять его всем пользователям бота. 
    
Уведомления пользователей:
     Если пользователь начал оставление заявки  и не завершил его, 
     то через 10 минут бездействия бот должен отправить пользователю сообщение “Ты забыл заполнить заявку!”
```
#### 1. Установка зависимостей
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

